# Tab Tweaks

Improvements and customization for Minecraft's Tab / Player List HUD

## Download

Downloads can be found on the [Modrinth page](https://modrinth.com/mod/tabtweaks).

## Features

- Tab Widget Color
- Move Tab List Down
- Move Tab List Below Bossbars
- Show Numerical Ping
- Scale Numerical Ping
- Hide Fake Ping
